Project Title

User Types
	user
	admin

Roles of Admin:
---------------------
	1.Open New Account

	2.Delete Existing Account

	3.View Specific or All Accounts

	4.Add Balance to a User Account

	5.Update System Email or Password

	6.View User Issues or Complaints

	7.Logout

Roles of User:
------------------
	1.Profile Picture Display

	2.Welcome Message

	3.View Personal Account Details include password or balance

	4.Update Personal Information

	5.Withdraw Money

	6.Transfer Money to Another Account

	7.View Transaction History

	8.Logout
